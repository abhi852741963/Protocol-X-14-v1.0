[1] Voice-Controlled Home Automation (2022)

Objective: Build a home automation system controlled by voice.

What went wrong: Incomplete voice recognition implementation, poor hardware relay response.

Lesson learned: Start small with one function before scaling, and test modules separately.

[2] AI Chatbot with Sentiment Analysis

Objective: Create a smart chatbot that adapts based on mood.

What went wrong: NLP model was unstable; responses were generic and repetitive.

Lesson learned: Use pre-trained models initially instead of training from scratch; understand limitations of sentiment detection.

[3] Bluetooth-Controlled Car (Arduino)

Objective: Build an RC car with Bluetooth commands.

What went wrong: Cheap Bluetooth module kept disconnecting; power supply wasn’t stable.

Lesson learned: Always test hardware in parts, and use reliable components for communication modules.

[4] Jarvis v0.5 (Old Build)

Objective: Create an assistant that reads out emails and weather.

What went wrong: Speech-to-text failed often; poor API integration.

Lesson learned: Don’t skip documentation; properly handle errors and timeouts with APIs.
